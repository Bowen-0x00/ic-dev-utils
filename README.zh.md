# chip-dev-toolbox
数字IC/处理器开发中的通用工具积累

## 📖 简介 | Overview
本仓库用于集中管理数字IC与处理器设计中常用的构建脚本与仿真工具，包括：
- 🧩 **Makefile 模板**：支持模块化编译、文件依赖自动生成等；
- 🐍 **脚本工具**：TCL/Python，用于生成 filelist、dump FSDP、仿真自动化等；
- ⚙️ **SystemVerilog 实用模块**：仿真专用 UART、watchdog、仿真终止控制器等。

**主要目标：**  
快速原型搭建 / 仿真环境配置 / 调试辅助。
